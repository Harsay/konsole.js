# konsole.js
Command line like application in browser.

Current version: 0.1

## How to use
``` javascript
// example
konsole.init("#id");
```
```
command arguments
```

## Available commands
| command | arguments | description |
| ------- | --------- | ----------- |
| help    | command   | prints information about given command |
| print   | text      | prints given text |
| version | -         | shows konsole.js version |
